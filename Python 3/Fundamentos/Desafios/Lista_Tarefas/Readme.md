# lista_tarefas
    Desafio realizado com intuito de praticar conhecimentos adquiridos sobre listas durante o curso de Python3.

    Neste desafio o objetivo foi realizar um cadastro simples de tarefas a serem executadas com um menu de opções onde o usuário pode interagir com o programa.

    Nele estão contidas as opções:
    - **Adicionar até cinco tarefas**
    - **Listar as tarefas adicionadas**
    - **Marcar como lida e consequentemente remover da lista**
    - **Sair e encerrar o programa.**
